눈에 보이는 내용을 만들기위해서 화면에 정보를 출력하는 디스플레이 태그가 어떤것들이 있고 어떤 역할을 하는지
body 태그 안에 들어가서 정보를 사용자에게 표기하는 태그

이러한 표기용 태그들을 카테고리별로 나눠서 수업한다
    1. 글자 및 서식
    2. 멀티 미디어
    3. 입력
    4. 기타

즉, 오늘 배울 내용은 body 태그 안에 들어감으로써 사용자에게 글자를 표기할 수 있는 태그
    h1
        제목을 표기하기위한 태그
        안에 적은 제목 텍스트를 큰 글자로 표기하는 태그
        작성법 - <h1>제목</h1>
        특이사항
            h2 ~ h6 까지 소제목을 위한 태그도 함께 존재
            단, h6까지 태그가 "이미" 존재하는것

    pre
        본문을 표기하기위한 태그
        문단을 표기하기위한 태그
        안에 적힌 내용을 "그대로" 화면에 추출하는 태그
        작성법 - <pre>내용</pre>

    hgroup
        제목과 본문을 묶음으로 만들어주는 태그
        작성법 - <hgroup> <h1>제목</h1> <pre>본문</pre> </hgroup>

서식 태그의 종류
    strong, b
        글자를 두껍게하는 서식 태그
        해당 내용만 글자가 두껍게 출력되도록 서식 지정
        작성법 - <strong>내용</strong>

    em, i
        글자를 기울이는 서식 태그
        해당 내용만 글자가 기울어져서 출력되도록 서식 지정
        작성법 - <em>내용</em>

    ins, u
        글자에 밑줄을 작성하는 태그
        해당 내용에만 글자밑에 선이 출력되는 서식 지정
        작성법 - <ins>밑줄</ins>

    del, s
        글자에 관통선을 작성하는 태그
        해당 내용에만 취소선을 출력하는 서식 지정
        작성법 - <del>취소</del>
    
    sup
        글자를 윗첨자로 변경하는 태그
        작성법 - <sup>내용</sup>

    sub
        글자를 아랫첨자로 변경하는 태그
        작성법 - <sub>내용</sub>

    ruby
        해당 태그 안에 적은 내용중에 알아보기 힘든 발음이 있다면 따로 위에 표기를하는 태그
        일본어의 후리가나를 위해 사용
        작성법 - <ruby>公差<rt>コシャ</rt></ruby>

    rt
        ruby 후리가나 태그 안에 적어서 위쪽에 적힐 내용을 표기하는 태그
        작성법 - <ruby>公差<rt>コシャ</rt></ruby>
    
    rp
        후리가나 태그가 동작되지 않는 환경에서도 동일한 기능을 보여주기위해서 대신 보여줄 내용을 작성하는 태그
        작성법 - <ruby>公差<rp>(</rp><rt>コシャ</rt><rp>)</rp></ruby>

    p
        아무런 기능 없이 단순하게 글자의 본문을 표기하는 서식 태그
        작성법 - <p>내용</p>

    br
        줄바꿈 태그
        본문 중간에 작성을하면 해당 내용 다음 내용은 다음줄로 줄바꿈을 하는 태그
        작성법 - <br/>
