HTML 규칙
    0. 파일명은 .html로 끝나야한다
    1. 첫줄은 <!DOCTYPE html> 로 시작한다
        HTML을 사용하는건 미지의 누군가에게 정보를 전달하고자할때 사용한다
        그러다보니 내가 전달하는 정보가 어떤 규격인지도 모르는 경우가 생긴다
        그래서 첫줄에 아래에 나오는 내용이 HTML 규칙으로 작성된걸 알려주는 접두사가 필요
    2. 모든 내용은 태그로 관리된다
        태그는 싱글라인, 멀티라인으로 나뉘며 정보를 규격화하는 형식
            <tag/> -> 싱글라인
            <tag></tag> -> 멀티라인
        작성되는 내용에서 중요한 부분 을 굵게 하고싶을때 확인이 안된다
        그래서 <bold>중요한 부분</bold>을 태그로 감싸고 선택할 수 있도록
        태그의 구조를 이용해서 정보를 규격화

        그래서 싱글라인은 따로 내부에 무언가 필요없이 정보를 표현할때 사용 -> image, video, ...
        멀티라인은 내부에 추가적인 텍스트(멀티미디어)를 선택한 후 표현할때 사용 -> text, ...
    3. 태그는 아무거나 사용이 가능한게 아니라 정해진대로 사용해야한다
        즉, 태그는 이름을 마음대로 지을 수 없고 정해진 이름이 존재한다
        그리고 정해진 이름의 태그에는 각각의 정해진 역할이 있다
    4. 태그는 일반적인 사용법을 넘어서 추가적으로 사용하는 속성을 이용해서 변주를 가한다
        <tag/> -> 언제나 동일한 내용
        <tag attribute/> -> 속성을 태그에 추가할 수 있고 추가된 내용에 따라 태그가 변하도록 구성이 가능하다
            <html lang="ko"></html> -> 웹페이지를 구성하는 태그
        그래서 태그가 여러개일때는
            <tag aName=aValue bName=bValue .../>
    5. HTML의 모든 태그는 html 태그안에 들어있고 화면 구성은 body 태그안에 웹 정보는 head 태그 안에 들어있다
        모든 태그를 작성할때 해당하는 태그의 목적에 따라서 위치가 정해져있다
        <html>
            <head>
                // 웹 정보를 표기하기위한 태그
            </head>
            <body>
                // 화면 구성을 표기하기위한 태그
            </body>
        </html>

        이렇게 세팅한 후 F5 클릭후 Web App(Chrome or Edge) 클릭 시 실행
        실행하는 화면으로 원하는대로 작성했는지 확인 가능
    6. 작성되는 태그 안/밖 의 글자들은 모두 단순 텍스트로 표현
        <tag attrib=value/> 이런식으로 속성으로 넣는 대상이 아니라면 
        작성되는 모든 글자는 태그의 안/밖에 "따로" 존재하고 그건 일반 텍스트로 표현

        그래서 글자에 "가공"을 가하고 싶다면 "태그의 기능"을 이용해서 가공
    7. HTML에서 일반 텍스트로 제공되는 모든 공백문자는 한개의 띄어쓰기로 변형
        "           Hello            " -> " Hello "
        아무거나 합치는게 아니라 동일한 위치에있는(이어서 작성하는) 공백문자들은 모두 한개의 띄어쓰기로 합쳐버린다
        공백문자 -> 띄어쓰기, 탭, 엔터 같이 보여지는것이 없는 글자

        의도적으로 여러개의 공백을 넣을 수 없게되었다
        그래서 HTML에는 기본적으로 규칙이 표현을 못하게 막은 글자들을 표현할 수 있도록
        &nbsp; 같이 특수한 모양의 글자도 존재한다


내가 first 라는 파일에 내 이름과 성별을 작성해서 보여줘야지
    1. first.html 파일 생성
    2. <!DOCTYPE html> 접두사 작성
    3. <html></html> 태그 작성
    4. <head></head> 태그 html 태그 안에 작성
    5. <body></body> 태그 html 태그 안에 작성
    6. 자신의 이름, 성별을 body 태그 안에 작성
